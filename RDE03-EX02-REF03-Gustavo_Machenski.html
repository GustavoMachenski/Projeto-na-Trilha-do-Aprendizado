HTML

É uma Linguagem de Marcação de Hipertexto baseada em tags criada na década de 1990 por Tim Berners-Lee para publicação de conteúdo em hipertexto, tais como, texto, imagem, áudio e vídeo, e que podem ser vinculados através de hyperlinks.

TAGS

As tags definem ao navegador a marcação de conteúdo.
As tags tem funcionalidades específicas. Sem o uso de tags o navegador exibe um texto planp

As tags podem possuir atributos, ou seja, são parâmetros que definem algun tipo de informação extra ao navegador

LISTAS

No HTML podemos utilizar 3 tipos de listas basicas, sendo elas:

  1. Não ordenadas
  2. Ordenadas
  3. Detalhadas

Cada umas delas tem uma sintaxe diferente, sendo elas na ordem:

Não ordenadas
< ul >
< li >
< li >
< /ul >
Ordenadas
< ol >
< li >
< li >
< /ol >
Detalhadas
< dl >
< dt >
< dd >
< dt >
< dd >
< /dl >
