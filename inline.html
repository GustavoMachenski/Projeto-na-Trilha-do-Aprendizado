O CSS é utilizado para estilizar os documentos HTML. Sua sintaxe é:

A três formas de se utilizar o CSS:

  • INLINE

    • Os códigos CSS são definidos dentro da tag HTML e afeta somente a tag que ele esta dentro.

  • INCORPORADO

    • O CSS incorporado é definido dentro da tag head e afeta todo o documento em que ele está.

  • EXTERNO

    • O código externo é definido em um arquivo externo e afeta todos os documentos linkados a ele.
