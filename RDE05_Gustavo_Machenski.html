Hyperlinks ou Link

É uma referencia dentro de um documento qeu se liga a outras partes dele mesmo ou a outros documentos, imagens, arquivos entre outros.

Sintese de Hyperlinks

  • Tag utilizada: < a >
  • Entrada de referencia href=""

Hiperlinks internos

Utilizado quando se deseja referenciar algo dentro da mesma pagina. Sua sistese é um pouco diferente:

Onde o name é utilizado na parte a ser referenciada e o href="#" é onde a palavra é definida para ser linkada.

Hperlinks Locais

Levam para outra pagina dentro de um mesmo site.

O href="" faz referencia a um arquivo html.
